# WoW-Prime Admin Panel

Клиентский Lua-аддон для удобного выполнения GM-команд в World of Warcraft 3.3.5a
на сервере **WoW-PRIME** (TrinityCore-based).

## Происхождение

Адаптация под TrinityCore форка [AzerothAdmin](https://github.com/55Honey/AzerothAdmin),
который в свою очередь является форком оригинального TrinityAdmin/MangAdmin.

## Установка

1. Скачать релиз
2. Распаковать в `Interface/AddOns/WoWPrimeAdminPanel/` в папке клиента WoW 3.3.5
3. Запустить клиент, активировать аддон на экране выбора персонажа

Требуется GM-доступ (SecurityLevel >= 1) на сервере.

## Использование

Открыть аддон командой `/wpa` или биндом по умолчанию.

## Совместимость

- Клиент: WoW 3.3.5a (build 12340)
- Сервер: TrinityCore 3.3.5
- Тестировался на сервере WoW-PRIME

## Лицензия

Соответствует лицензии исходного AzerothAdmin / TrinityAdmin.
